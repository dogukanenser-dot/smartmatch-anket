# SMART MATCH – Anket Değerlendirme (Word → Skor → Öneri)

Bu klasör Streamlit uygulamasıdır.

## Seçenek 1: Streamlit Community Cloud (Tarayıcıdan Kullanım)
1. Bu klasörü GitHub'a yükleyin (public veya private).
2. Streamlit Community Cloud'da “New app” ile repo'yu seçin.
3. Main file: `app.py`
4. Deploy edin. Size bir link verir.

## Seçenek 2: Lokal (Python ile)
Python 3.10/3.11 önerilir.
- `pip install -r requirements.txt`
- `streamlit run app.py`

Not: Eğer Windows/Python sürüm uyuşmazlığı yaşarsanız, “Seçenek 1” en sorunsuz yoldur.
